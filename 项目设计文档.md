# 单词消消乐 - 项目设计文档

## 项目概述

单词消消乐（Word Match）是一款基于 Web 的单词配对学习游戏，支持四级、六级、托福、GRE 四类词库，采用"先学后玩"模式：先通过闪卡学习单词，再进行消消乐配对游戏。

技术栈：纯 HTML + CSS + JavaScript 单文件实现，无需后端。

---

## 已实现功能

### 核心玩法
- 词库选择（CET4 / CET6 / TOEFL / GRE）
  - CET-4: 4412 词 | CET-6: 5506 词 | TOEFL: 103 词 | GRE: 108 词
- 每组学习词数可调节（增减按钮）
- 学习阶段：闪卡翻转 + 自我评估（不认识/模糊/认识）
- 游戏阶段：英文-中文配对消消乐，带计时、连击、得分
- 道具系统：提示、+10s、重排
- 结算页面：星级评价、金币/宝石奖励、单词回顾

### 学习数据统计
- 学习统计面板：饼图(掌握度分布)、词库进度条、累计学习天数/正确率/最高连击/累计金币
- 词书浏览/生词本：按掌握度筛选、搜索、单词详情展开、发音、收藏功能

### 积分经济系统
- 金币/宝石货币、商店购买道具、等级体系、每日打卡奖励
- 累计总分、累计正确率、最高连击、累计金币等扩展数据持久化

### 记忆系统
- 基于艾宾浩斯遗忘曲线的记忆追踪（MemorySystem）
- 单词熟悉度分级：新词 → 陌生 → 模糊 → 熟悉 → 掌握 → 熟练
- 学习数据本地持久化（localStorage）

### 音效系统
- 基于 Web Audio API 的合成音效（SFX）
- 包含：选中、正确配对、错误、连击、翻卡等音效

### 单词发音（2026-06-27 新增）
- 技术方案：使用浏览器内置 Web Speech API（SpeechSynthesis），无需外部音频文件
- 学习卡片：音标下方显示🔊发音按钮，点击播放单词朗读；每张新卡片展示时自动发音一次
- 结果回顾：每个单词左侧显示🔊小喇叭图标，点击可重复听发音
- 发音参数：en-US 英语，速率 0.85（略慢于正常，便于学习）

## 待讨论/优化项

### 连连看游戏机制的局限性（2026-06-27 讨论）
- **问题**：词库扩充至 4000+ 词后，连连看界面拥挤（面板 max-height 390px，方块 min-height 50px），每轮最多 20 词但实际体验 10 词已很挤
- **核心矛盾**：需要提高每轮学习量以加快进度，但连连看 UI 不支持更多词同时显示
- **备选方案讨论**：
  - 多模式混合（判断对错 + 四选一 + 拼写等，按难度自动切换）
  - 替换为选择题（经典高效，不受词数限制）
  - 单词雨（下落消除，有紧张感）
  - 保留连连看但缩小规模（限 6 对，配合其他游戏形式）
- **结论**：待用户确认方向后实施
